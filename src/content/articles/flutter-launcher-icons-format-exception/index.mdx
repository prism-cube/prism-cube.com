## 環境

```
Flutter 2.10.0
Dart 2.16.0
```

```yaml title="pubspec.yaml"
dependencies:
  flutter_launcher_icons: ^0.9.2
```

## 対処方法

flutter_launcher_iconsをインストールし、

```shell
flutter pub run flutter_launcher_icons:main
```

を実行すると

```
✓ Successfully generated launcher icons
Unhandled exception:
FormatException: Invalid number (at character 1)
```
上記のエラーが発生した。

[https://github.com/fluttercommunity/flutterlaunchericons/issues/316](https://github.com/fluttercommunity/flutterlaunchericons/issues/316)

上記リンクを参考に、pubspec.yamlを修正し、

```yaml title="pubspec.yaml"
flutter_launcher_icons:
  git:
    url: https://github.com/Davenchy/flutter_launcher_icons.git
    ref: fixMinSdkParseFlutter2.8
```

再度実行するとエラーなくアイコンを生成することができる。
